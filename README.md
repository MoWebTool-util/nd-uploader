# nd-uploader

[![spm version](http://spmjs.io/badge/nd-uploader)](http://spmjs.io/package/nd-uploader)

> 

## 安装

```
$ spm install nd-uploader --save
```

## 使用

```js
var Uploader = require('nd-uploader');

使用时，需手动配置vendor下的Uploader.swf
Uploader.create({
	swf: ''//swf地址
})
// use nd-uploader
```
